Contact

You can contact the developers of Schoolsplay in various ways:

  • Subscribe to the mailinglist (This will get you usually the fastest reply!)
  • Send a mail to the developers: stas.zytkiewicz@gmail.com or
    chris.van.bael@gmail.com
  • If you want to report a bug, you can do so at the Schoolsplay bugtracker
  • If you want to request a feature, please submit it via
    the Schoolsplay feature requests

If you contact us for a problem you encountered, please fill in following template:

Platform: (Windows, Linux, OSX)
Application: (Childsplay, Schoolsplay, Cognitionplay)
Version:
Activity:
Language:
Description of problem:

This will save us time and will give you a faster response!
